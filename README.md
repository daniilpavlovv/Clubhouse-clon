# Clubhouse clon

<img src="https://i.ibb.co/GQGdXvZ/screencapture-localhost-3000-rooms-2-2021-08-06-15-13-16.png" alt="screencapture-localhost-3000-rooms-2-2021-08-06-15-13-16" border="0">

**Stack:**

Frontend:
- NextJS / ReactJS
- TypeScript
- Redux
- simple-peer (WebRTC)
- Socket.IO
- Immer
- Axios

Backend:
- NodeJS
- Express
- Sequelize
- Passport
- JWT
- Multer
- PostgreSQL

**Launch:**

- Create a "clubhouse" database in PostgreSQL
- Running migrations
```
yarn migrate
```
- Run development server
```
yarn dev
```
- Run NodeJS server
```
yarn server
```
