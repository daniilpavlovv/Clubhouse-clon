# Clubhouse clon

<img src="https://i.ibb.co/DtVZm89/screencapture-192-168-31-75-3000-2021-08-06-13-50-48.png" alt="screencapture-192-168-31-75-3000-2021-08-06-13-50-48" border="0">

**Stack:**

Frontend:
- NextJS / ReactJS
- TypeScript
- Redux
- simple-peer (WebRTC)
- Socket.IO
- Immer
- Axios

Backend:
- NodeJS
- Express
- Sequelize
- Passport
- JWT
- Multer
- PostgreSQL

**Launch:**

- Create a "clubhouse" database in PostgreSQL
- Running migrations
```
yarn migrate
```
- Run development server
```
yarn dev
```
- Run NodeJS server
```
yarn server
```
