**Stack:**

Frontend:
- NextJS / ReactJS
- TypeScript
- Redux
- simple-peer (WebRTC)
- Socket.IO
- Immer
- Axios

Backend:
- NodeJS
- Express
- Sequelize
- Passport
- JWT
- Multer
- PostgreSQL

**Launch:**

- Create a "clubhouse" database in PostgreSQL
- Running migrations
```
yarn migrate
```
- Run development server
```
yarn dev
```
- Run NodeJS server
```
yarn server
```
